# Today I Learned

- `git init`
    - 새로운 git 레포 생성
- `git status`
    - 브랜치, 커밋여부 등등 확인 가능

- `git add 파일명`
    - add를 하기 전에는 각 파일들이 옆에 U로 표시된다 (Untracked)
    - add를 한 파일은 옆에 A로 표시된다 (index added)
    - `.` -> 현재 위치의 모든 파일
        

- `git commit -m "파일명"`
    - `commit`
    - `-m` -> 메세지
    - "파일명"의 이름으로 커밋

- `git log`
    - 커밋을 한 로그(기록) 표현
    - `--online`
        - 각각의 커밋 정보를 간단하게 표현
    - `--graph`
        - 브랜치까지 시각적으로 표현해주는 옵션
