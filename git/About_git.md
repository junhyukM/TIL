# Git이 뭘까?

## Git을 이용한 버전 관리란?
### DVCS(분산 버전 관리 시스템) 
- 수정사항을 별개로 저장하거나 하는 기존의 방식과 다르게
- 사진을 찍는 것에 비유할 수 있는 개념으로, 데이터를 사진과 같이 취급하여
- 데이터를 다룬다
- 관리되고 있는 데이터를 개인 로컬에 받아 서버가 정상동작 하지 않을때도 데이터를 사용하는 것에 문제가 생기지 않는다는 장점이 있다

- 이러한 방식으로 동시다발적 작업과 단순한 버전관리에 용이하기에 지금까지 가장 많은 사용자를 가지고 있다
